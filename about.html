Programming

So far in my foray into programming, there have been several classes that have stuck out:

  • Data Structures: In this class I learned how to program almost every important data structure from the ground up in Java. These included linked lists, queues, stacks, and hash tables to name the big ones. I also got to learn about time complexity and sorting algorithms. Knowing that whatever I use in my program has an effect on either the space it takes up or the time it takes to complete was really eye-opening for me.
  • Systems Programming and Operating Systems: These two classes allowed me to see the inner workings of computers. We learned about UNIX and terminal in Systems Programming and mainly threads, semaphores, and memory management in OS. The most important takeaway from these two classes was learning C as well as UNIX system operations.

3D Printing

Since I am generally indisposed at home with spare time, 3D printing is something that I have fallen in love with due to the more set and forget nature of it. I think that this hobby is one of the best things I could have discovered because it combines my love for technology with the (almost) instant gratification of getting to hold a physical object in your hand that you, or somebody else, designed. These objects can be practical in nature or completely decorative, which is what I mostly stick to. I love this hobby so much, in fact, that I even plan to make a page on this website dedicated to hosting STL files that I have made. Also, if you haven't noticed, my Prusa Mk3 is the background image!
