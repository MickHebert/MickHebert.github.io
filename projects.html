Password Manager

A very primitive password manager that generates a random string for the user to use as a password. Currently only saves as plaintext and should not be used for sensitive information! Here is a link to its GitHub repository

Written quickly in Python to generate random stings of characters to be used as passwords. Is able to save them in plaintext along with what website they were made for. This is by no means a safe program and I mainly made it to mess around in Python. However, I plan to add some functionality to it that will make it more useful and secure.
